# 快速启动指南

## 方案一：使用浏览器原生语音识别（推荐快速测试）

### 步骤 1: 直接打开演示页面

双击打开 `Voice/demo.html` 文件，即可在浏览器中体验语音输入功能。

### 步骤 2: 测试功能

1. 点击蓝色麦克风按钮
2. 允许浏览器访问麦克风
3. 开始说话（中文）
4. 查看实时识别结果
5. 再次点击按钮停止录音并发送

### 步骤 3: 集成到现有项目

如果需要集成到医药助教项目中，参考 `INTEGRATION_GUIDE.md` 文档。

---

## 方案二：使用 React 项目（推荐开发使用）

### 步骤 1: 安装依赖

```bash
cd Voice
npm install
```

### 步骤 2: 启动开发服务器

```bash
npm run dev
```

### 步骤 3: 访问应用

浏览器会自动打开 http://localhost:3001

---

## 方案三：集成到豆包 API（生产环境）

### 步骤 1: 获取 API 凭证

访问火山引擎控制台，获取：
- APP ID
- Access Token

### 步骤 2: 配置环境变量

在项目根目录的 `.env` 文件中添加：

```env
VOLC_APP_KEY=your_app_key_here
VOLC_ACCESS_KEY=your_access_key_here
```

### 步骤 3: 实现后端 API

参考 `INTEGRATION_GUIDE.md` 中的"升级到豆包语音识别 API"部分。

### 步骤 4: 使用豆包语音组件

将 `VoiceInput` 替换为 `VoiceInputDoubao` 组件。

## 常见问题

### Q: 浏览器提示不支持语音识别怎么办？

A: 使用 Chrome 或 Edge 浏览器，并更新到最新版本。

### Q: 麦克风权限被拒绝怎么办？

A: 在浏览器设置中允许网站访问麦克风，或检查系统权限设置。

### Q: 识别准确度不高怎么办？

A: 
- 减少环境噪音
- 清晰、缓慢地说话
- 调整麦克风位置
- 考虑使用豆包 API 提高准确度

### Q: 如何在现有项目中使用？

A: 参考 `INTEGRATION_GUIDE.md` 文档，有两种集成方式：
1. 直接复制组件文件
2. 使用独立演示页面
