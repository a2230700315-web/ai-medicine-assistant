# AI药学助手系统 - 已实现功能清单

## 一、系统架构概览

### 1.1 用户角色体系
| 角色 | 权限范围 |
|------|---------|
| 店员 | 学习中心、案例练习、模拟题、真题考试 |
| 店长 | 店员全部权限 + 门店管理 |
| 总部管理员 | 全部权限 + 总部看板 |

### 1.2 技术栈
- **前端**: React 18 + Vite + Tailwind CSS
- **图表**: ECharts
- **AI**: 火山引擎(豆包)大模型
- **后端**: FastAPI
- **部署**: Vercel
